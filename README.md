# Proyecto en Astro para realizar pruebas del backend

Para iniciar el proyecto:
1. Clonar repositorio
2. Navegar a la raiz del proyecto e instalar dependencias:

    ```sh
    npm install
    ```
3. Ejecutar servidor local:

    ```sh
    npm run dev
    ```
